# 试验目录说明

本目录用于碰撞检测算法原型验证与性能思路实验，不直接作为最终结论输出目录。

## 目录定位

- 快速验证“想法是否可行”。
- 对比不同碰撞判定策略的正确性与速度。
- 为 `../正式代码/` 提供可迁移的算法模块。

## 主要文件

- `图形碰撞相交判定.py`
  - 基于分离轴定理（SAT）的矩形相交判定示例。
  - 用于验证几何判定逻辑与边界情况处理。
- `碰撞相交.py`
  - 基于 shapely 的相交判定示例。
  - 用于与 SAT 结果交叉验证，辅助排查几何实现错误。
- `动态四叉树.py`
  - 空间索引试验脚本。
  - 用于验证“候选对筛选加速”是否具备收益。
- `untitled2.py`
  - 历史试验脚本，主要用于保留实验轨迹。

## 典型实验流程

1. 构造最小可复现实验（少量线段、少量时间步）。
2. 先验证判定正确性，再比较时间开销。
3. 记录通过实验的参数与假设条件。
4. 将稳定方案迁移到 `../正式代码/`。

## 迁移准入标准（建议）

- 正确性：与基准方法在关键样例上一致。
- 稳定性：多次运行结果一致。
- 可维护性：接口清晰，参数可配置。
- 可复现性：有明确运行方式和输出说明。

## 注意事项

- 本目录参数通常简化，不代表正式评测参数。
- 试验脚本若引入第三方依赖，请在代码注释或 README 里说明版本要求。
