# 2-code 目录说明

本目录是 2024 国赛 A 题的代码主目录，覆盖“正式求解、过程脚本、算法试验”三个层次。

## 子目录职责

- `正式代码/`：最终复现入口，输出可直接用于展示与论文的数据/图像。
- `script/`：问题拆解脚本与中间计算脚本，用于推导、调参和过程验证。
- `试验/`：算法原型和碰撞判定实验，用于验证新思路可行性。

## 推荐使用流程

1. 先在 `试验/` 验证新判定逻辑是否正确。
2. 将稳定方案迁移到 `正式代码/`，统一参数和输出格式。
3. 运行正式脚本后，从 `viz_output/` 获取图、表和摘要结论。

## 一键复现（推荐）

```bash
cd 正式代码
python 第二问_优化可视化展示.py
```

运行后会在 `正式代码/viz_output/` 生成：

- `collision_cover.png`（封面图）
- `dragon_motion.gif`（过程动画）
- `collision_timeline.png`（时间线）
- `collision_frame.png`（关键帧）
- `collision_dashboard.html`（交互页面）
- `collision_metrics.csv`（逐帧指标）
- `summary.json`（结构化结论）

## 协作规范建议

- 正式结论必须以 `正式代码` 目录输出为准。
- 草稿脚本可以快速迭代，但建议在文件头注明“是否可用于最终结论”。
- 若新增脚本，请同步在对应子目录 README 中补充用途说明与运行方法。
