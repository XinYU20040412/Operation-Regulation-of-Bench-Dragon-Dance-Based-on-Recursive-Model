# 图片目录说明

本目录用于管理论文与展示所需的图像资源（位图/矢量图）。

## 目录职责

- 存放人工整理后的论文插图素材。
- 保存高质量矢量图，保证打印与放大时清晰度。
- 作为 `2-code/正式代码/viz_output/` 自动生成图像的“精选汇总区”。

## 当前文件

- `龙头三百秒内位置.eps`：龙头轨迹相关矢量图。

## 推荐命名规范

- 建议格式：`问题编号_图编号_主题_版本.扩展名`
- 示例：`Q2_Fig03_碰撞时间线_v2.eps`
- 若图来自脚本自动导出，建议在名称中保留来源信息（如 `from_viz_output`）。

## 格式选择建议

- 论文终稿优先：`EPS` / `PDF` / `SVG`（矢量格式）。
- 网页展示优先：`PNG` / `GIF`（加载兼容性更好）。
- 不建议在终稿中使用低分辨率截图，避免打印模糊。

## 与代码目录的协作方式

1. 在 `2-code/正式代码` 运行主脚本生成最新图像。
2. 从 `viz_output` 中挑选需要的图复制到本目录并重命名。
3. 在论文中引用本目录图像，保持引用路径稳定。

## 质量检查清单

- 线条和文字在 A4 打印下是否清晰。
- 坐标轴单位、图例、时间刻度是否完整。
- 图中关键事件（如首次碰撞）是否有明确标注。
